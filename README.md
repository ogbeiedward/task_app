# Task App

A Flutter task management application built as part of the **Application of Artificial Intelligence in Mobile Devices** course (Lab 3).

**Author:** Edward Ogbei

## Features

- Add new tasks with validation
- Edit existing tasks
- Delete tasks
- Mark tasks as complete with checkbox
- Completed tasks shown with strikethrough

## Project Structure

```
lib/
├── data/
│   └── models/
├── model/
│   └── task.dart
├── presentation/
│   ├── screens/
│   ├── widgets/
│   └── my_home_page.dart
└── main.dart
```

## Getting Started

1. Make sure Flutter is installed
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to launch the app
